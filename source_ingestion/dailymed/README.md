# DailyMed Experiment

Purpose:

- Evaluate DailyMed as a source-linked labeling reference for recognized
  products and ingredients.

Initial experiment:

- Link normalized products to labeling references.
- Preserve label source URL and retrieval date.
- Do not treat labeling text as a complete interaction reasoning engine.

Required output fields:

- source metadata
- source record identifier
- product or label display name
- linked normalized identifier if available
- mapping confidence
- review status
